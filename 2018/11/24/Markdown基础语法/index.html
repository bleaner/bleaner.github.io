Markdown基础语法

  • 2018-11-24

标题

在行首加入1-6个#，表示1-6级标题，例如：

#这是一级标题

##这是二级标题

###这是三级标题

注意：#后有一个空格

文本样式

加粗：

在需要加粗的文字前后各加两个星号或者_底线符号，前后符号要匹配且符号和字之间不能有空格。例如:
\*Hello**或__Hello__的效果是Hello

斜体

在需要倾斜的文字前后各加一个星号或者_底线符号。例如：
\Hello*或_Hello_的效果是Hello

斜体加粗

前后各加三个星号或者_底线符号。例如：
\**Hello***效果是Hello

高亮

前后各加一个及以上的“`”反引号（Esc键下方）。例如：
`Hello`的效果是Hello

删除线

前后各加两个~波浪号（Esc键下方）。例如：
~~Hello~~的效果是Hello

分割线

在单独的一个文本行内用三个及以上的星号、_底线符号来建立分割线（符号间可以有空格）。例如：
\**

段落和换行

  1. 一个Markdown段落由一到多个连续的文本行组成，段落前后要有一个以上的空行（若某一行只包含空格和制表符也被视为空行）。
  2. 换行：在一个文本行末键入两个及以上的空格或制表符（Tab键）后回车（Enter键）即可换行。另外，由于Markdown兼容部分HTML语法，也可以用<br>来换行。

    列表

  1. 无序列表
    每个文本行前使用星号、+加号、-减号来标记。例如：
    \     red
    * yellow
    * blue
    效果是：
  • red
  • yellow
  • blue
  1. 有序列表
    使用阿拉伯数字接着一个英文句点。例如：
    1. red
    2. yellow
    3. blue
    效果是：
  2. red
  3. yellow
  4. blue

    表格

  1. 每个表头或表格单元之间用“|”来隔开。表头下要有-|-|-，有几个表头就有几个“-”减号。
  2. 左对齐：默认左对齐，或在“-”左侧加“:”表示左对齐。
  3. 右对齐：在“-”右侧加“:”表示右对齐。
  4. 居中对齐：在“-”两侧都加“:”表示居中对齐。
    例如：
姓名\ 班级\ 学号
张三 1 10001
李四 2 10002
王五 3 10003

效果是：

姓名 班级 学号
张三 1 10001
李四 2 10002
王五 3 10003

注意：表格前要有空行

区块引用

引用以>来表示，引用中支持多级引用、标题、列表、代码块、分割线等常规语法。
区块引用可以嵌套，只要根据层次加上不同数量的 >，>的个数表示引用的级数。

代码区块

  1. 每行代码前缩进四个空格或一个制表符即可建立代码区块。
  2. 在需要插入的代码区块的前一行及后一行各使用三个“`”反引号。例如：
    ```
    #include<stdio.h>
    int main()
    {
    printf(“Hello word!”);
    }
    ```

注意：一个代码区块会一直持续到没有缩进的那一行（或是文件结尾）

添加超链接

[链接名称](链接地址)
例如：[bleaner’s blog](https://bleaner.github.io\)
效果是：
bleaner’s blog

链接图片

语法：
![图片alt](图片链接 “图片title”)
图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加。
示例：
![Alt text](./jordan-mcgee-1175109-unsplash.jpg)
