防火墙配置

目前我们电脑开启IIS服务，就相当于一台具有IIS服务的web服务器，我们要保证其他机器能够访问我们的网站，就必须考虑防火墙的问题。
防火墙和网络保护 - 高级设置

我们可以直接关闭防火墙，但是这是很不安全的措施
我们可以通过设置防火墙的出入站规则（ACL），对网站访问权限进行设置
入站规则：新建入站规则，选择对端口进行设置，我们可以设置为允许任何对于81端口的请求。
